KrishiAI

AI-powered crop yield prediction and recommendation system using machine learning and environmental data analysis.

Problem

Farmers rely on traditional methods for crop decisions, leading to poor yield predictions. Existing systems use limited parameters and fail to provide accurate, data-driven recommendations.

Gap & Solution

No system effectively compares multiple ML models before prediction. KrishiAI solves this by evaluating multiple models and selecting the best one using performance metrics.
