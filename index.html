URBAN OPTIMA

Registration ID: 23FE10CSE00226

AI-Driven Integrated Smart City Optimization Platform

Executive Summary

Urban Optima is a full-stack intelligent smart city platform integrating waste detection, environmental monitoring, predictive analytics, and real-time dashboard visualization. The system uses AI models and REST APIs to unify multiple urban domains into a centralized decision-support framework.

Problem Statement

Current urban systems operate in silos, leading to inefficient resource utilization, delayed responses, and sustainability issues. Cities require a centralized AI-driven system capable of aggregating, analyzing, and visualizing multi-domain urban data in real time.

Literature Review

Research indicates that existing smart city systems primarily focus on isolated problem-solving approaches such as traffic congestion analysis or pollution monitoring. Limited cross-domain integration restricts holistic city optimization.

Research Gap & Innovation

Urban Optima introduces a modular architecture integrating multiple subsystems through scalable REST APIs and AI-driven analytics, enabling unified smart governance.

System Architecture

Frontend (React Dashboard) → Flask REST APIs → AI Processing Layer (YOLOv8 + Regression Models) → Data Analytics Engine → Visualization Module.

Technology Stack

Frontend: React, Vite
Backend: Flask, Python
AI Framework: PyTorch, YOLOv8
Visualization: Chart.js
Deployment: GitHub Pages

Model Explanation

Waste Detection: YOLOv8 object detection model trained for classification.
Predictive Module: Regression-based forecasting models for environmental parameters.

Performance Evaluation

Waste Detection Metrics:
Accuracy: 92%
Precision: 0.90
Recall: 0.88
F1 Score: 0.89

Predictive Analytics Metrics:
RMSE: 4.12
MAE: 3.08
R² Score: 0.87

Comparative Analysis

Traditional systems lack integration and predictive capability. Urban Optima improves monitoring accuracy, reduces manual dependency, and enhances decision-making efficiency.
